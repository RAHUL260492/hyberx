# HyberX — Digital Growth Agency

Single-page site for HyberX. Everything lives in `index.html` plus the image assets in this folder.

## Deploy
Upload all files to any static host (Netlify, Vercel, GitHub Pages, cPanel). No build step.

## Local preview
Serve over HTTP (YouTube embeds do not work from file://):

    python3 -m http.server 8080

then open http://localhost:8080
